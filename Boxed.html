Boxed!

  • - Platform: Desktop Browser
  • - Language: Javascript
  • - Duration: 1 month
  • - Initial Completion: December 2019
  • - Last Update: March 2020
  • - Team Size: 1
  • - Technologies Used: p5.js, p5.play

As part of a university course, I was tasked with creating an interactive experience using the p5.js library plus one related library of my choice. Naturally, I used this opportunity to make a game. The result of this project: Boxed!

Boxed! is a simple game: move the mouse cursor to direct the green box around the screen in order to avoid obstacles. Using a procedural generation structure of my own design, the level is generated on the fly! The player's score is a direct measure of how long they were able to keep going through the level.

Play Boxed!

Project Goals

  • - Create a game using only Javascript libraries and pictures of squares
  • - Make a procedural generation system to generate levels during gameplay
